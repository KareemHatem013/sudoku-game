# sudoku-game
A simple sudoku game with both 6x6 and 9x9.The GTK stuff and design could be better than that but it's just a beginning.

Features:<br>
1- Both 6x6 and 9x9.<br>
2- Randomization<br>
3- Save<br>

Dependencies:<br>
gcc<br>
make<br>
gtk3<br>

```shell
# Creating an executable file
make
```
